# NextStationBudapest (JavaScript / Web)

## Overview
A web-based puzzle game inspired by the "Next Station" board game series, themed around the Budapest metro system. The project focuses on complex grid manipulation and state management using vanilla JavaScript.

## Technical Implementation
* **Dynamic Grid Logic:** Implemented a coordinate-based system to handle tile placement and path validation.
* **State Management:** Developed a custom engine to track game progress, available moves, and scoring logic without external frameworks.
* **DOM Manipulation:** High-frequency UI updates achieved through efficient DOM traversal and event handling.
* **CSS Grid & Flexbox:** Used advanced CSS layouts to ensure a responsive and visually accurate representation of the game board.

## Key Features
* Interactive tile placement.
* Path-finding logic (checking connections between stations).
* Scoring system based on metro line complexity.

## Technologies Used
* JavaScript (ES6+)
* HTML5
* CSS3 (Grid/Flexbox)
